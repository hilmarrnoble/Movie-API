API Endpoints

Request URL Response
GET /movies Returns a list of all movies in the database.
GET /movies/:title Returns movie details (description, genre, director, image URL) for the given title.
GET /genres/:name Returns genre details, including a brief description of the genre.
GET /directors/:name Returns director information, including bio and years of birth/death.
POST /register Registers a new user with the provided username, email, and password.
PUT /users/:id Updates a user's information (username).
POST /users/:id/favorites Adds a movie to a user's list of favorites.
DELETE /users/:id/favorites Removes a movie from a user's list of favorites.
DELETE /users/:id Deregisters a user and removes their account.
